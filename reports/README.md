# VelocityBench — Reports

This directory contains benchmark and QA reports generated by the test suite.

## Naming Convention

| Pattern | Generator | Description |
|---------|-----------|-------------|
| `bench-sequential-YYYY-MM-DD.json` | `make bench-sequential` | Sequential isolation benchmark results (JSON) |
| `bench-sequential-YYYY-MM-DD.md` | `make bench-sequential` | Sequential isolation benchmark results (Markdown) |
| `qa-verification-YYYY-MM-DD.json` | `python tests/qa/framework_validator.py` | Framework QA validation results (JSON) |
| `qa-verification-YYYY-MM-DD.md` | `python tests/qa/framework_validator.py` | Framework QA validation results (Markdown) |

## Archiving Policy

- Only the **most recent** report of each type lives at the root of `reports/`.
- Older reports are moved to `reports/archive/YYYY-MM/` (year-month of the report date).
- Archive is retained in git for historical comparison.

## Generating Reports

```bash
# Sequential isolation benchmark (canonical)
make bench-sequential
# or with options:
make bench-sequential DURATION=30 CONCURRENCY=40

# QA framework validation
python tests/qa/framework_validator.py
```

## Archive

Older reports are in `reports/archive/`:

- `archive/2026-02/` — Reports from February 2026
