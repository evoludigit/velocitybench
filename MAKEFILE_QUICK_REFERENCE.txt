🎯 CUSTOM GENERATION

  make blog-pattern PATTERN=<id>
    Generates tutorial (beginner) for specific pattern
    Example: make blog-pattern PATTERN=fraisier-deployment-orchestration

  make blog-pattern PATTERN=<id> DEPTH=<depth>
    DEPTH: beginner | intermediate | advanced
    Example: make blog-pattern PATTERN=fraisier-deployment-orchestration DEPTH=advanced

  make blog-pattern PATTERN=<id> TYPE=<type>
    TYPE: tutorial | troubleshooting | reference | comparison
    Example: make blog-pattern PATTERN=fraisier-deployment-orchestration TYPE=troubleshooting

🚀 QUICK START EXAMPLES

  # Generate one pattern (all blog types)
  $ make blog-fraisier

  # Generate intermediate tutorial for one pattern
  $ make blog-pattern PATTERN=webhook-event-driven-deployment DEPTH=intermediate

  # Generate troubleshooting guide
  $ make blog-pattern PATTERN=git-provider-abstraction TYPE=troubleshooting

  # Generate all blogs from all patterns
  $ make blog-all

  # Clean and regenerate
  $ make blog-clean && make blog-all
