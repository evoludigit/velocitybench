### Test Discovery

All tests are discoverable by their respective test runners:

**Python Frameworks**:
```bash
pytest frameworks/*/tests -m security
pytest frameworks/*/tests -m security_injection
```

**TypeScript/Node.js Frameworks**:
```bash
npm test -- --testNamePattern="Security"
vitest --grep="Security"
```

**Go Frameworks**:
```bash
go test ./... -run TestSecurity
```

**Rust Frameworks**:
```bash
cargo test security
```

**Java Frameworks**:
```bash
gradle test --tests "*Security*"
mvn test -Dtest=Security*
```

**Ruby Frameworks**:
```bash
ruby -Itest test/test_security_*.rb
rails test test/test_security_*.rb
```

**PHP Frameworks**:
```bash
phpunit --filter "Security"
```

**C# Frameworks**:
```bash
dotnet test --filter "FullyQualifiedName~Security"
```

**Scala Frameworks**:
```bash
sbt "testOnly *Security*"
```
